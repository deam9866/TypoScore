TypoScore ist ein Kompositionstool, welches es ermöglicht, Wörter und Texte in einem Notensystem einzugeben und die musikalische Repräsentation davon abzuspielen. Jeder Buchstabe erzeugt dabei einen unterschiedlichen Sound, die Tonhöhe wird durch die vertikalen Position der Buchstaben bestimmt. Das Ziel der Anwendung ist es, Zusammenhänge von Typografie und Musik bzw. musikalischer Notation zu entdecken, indem Musik als Alternative zur Sprachausgabe von Schrift genutzt wird.

Layout

Der Eingabebereich besteht aus einer beliebigen Anzahl von Notensystemen, in die einzeln Text eingegeben werden kann. Über die Buttons System hinzufügen und System entfernen kann die Anzahl individuell bestimmt werden, es wird jedoch immer mindestens eine Zeile angezeigt. Um die Texteingabe zu ermöglichen muss ein Notensystem ausgewählt sein. Dies geschieht über einen Klick in den Bereich der jeweiligen Notenlinien.

Textpositionierung

Der Punkt am Anfang jeder Zeile zeigt an, an welche Position geschrieben wird. Diese Position kann über die Pfeiltasten hoch und runter gesetzt werden. Um einen Buchstaben manuell zu verschieben, kann er durch Anklicken ausgewählt werden. Nun kann ebenfalls durch die Pfeiltasten die Position (Pfeil hoch bzw. runter) bzw. Größe (Pfeil links bzw. rechts) eingestellt werden.

Wiedergabe

Über den Pfeil am Ende einer Zeile kann die entsprechende Zeile abgespielt werden. Alternativ können mit dem Button Play links alle Zeilen nacheinander abgespielt werden.

Zufallsmodus

Neben der manuellen Positionierung können auch zufällige Positionen gewählt werden. Durch ein Klick auf den Button Zufällige Noten wird jedem Buchstaben eine Note zugewiesen. Über den Slider darunter kann der Bereich der Streuung eingegrenzt werden. Über den Button Noten zurücksetzen werden alle Buchstaben wieder auf eine Linie gesetzt.

Exportieren

Die Buttons Text speichern und Sound speichern dienen zum Speichern des Textes/Notensatzes und der Melodien. Letzteres funktioniert dabei nur für die zuletzt abgespielte Melodie. Um Text zu löschen kann innerhalb einer Zeile die Rück-Taste genutzt werden oder der Button Text löschen um den Text für alle Zeilen zu löschen.
